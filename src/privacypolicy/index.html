Privacy Policy for KimGeupSik Discord Bot

1. Introduction

This Privacy Policy explains how the KimGeupSik Discord Bot ("Bot") collects, uses, and shares your information.

2. Data Collection

The Bot collects the following data:

  • School ID and educational office ID associated with your user ID or guild ID.
  • Command logs including timestamp, channel ID, guild ID, user ID, command used, and prompt.

3. Purpose of Data Use

School information is used solely to provide meal information. Log data is used for Bot management and security purposes.

4. Data Storage and Retention

Data is stored on private servers and retained until a deletion request is made by the user.

5. Data Sharing

School and educational office IDs are used with the educational office's API to retrieve meal information. No other third-party sharing occurs.

6. User Rights

Users can request the deletion of their data by contacting us.

7. Changes to Privacy Policy

Any changes to this policy will be communicated via DM or guild messages.

8. Contact Information

For inquiries regarding this policy, contact bob@youngzheimer.com.

김급식 디스코드 봇 개인정보 처리방침

1. 소개

본 개인정보 처리방침은 김급식 디스코드 봇(이하 "봇")이 사용자 정보를 수집, 사용, 공유하는 방법을 설명합니다.

2. 데이터 수집

봇은 다음과 같은 데이터를 수집합니다:

  • 학교 ID와 교육청 ID를 사용자 ID 또는 길드 ID와 함께 저장.
  • 명령어 로그 (사용 시간, 채널 ID, 길드 ID, 사용자 ID, 사용한 명령어, 프롬프트 등).

3. 데이터 사용 목적

학교 정보는 급식 정보를 제공하기 위해 사용됩니다. 로그 데이터는 봇 관리 및 보안을 위해 사용됩니다.

4. 데이터 저장 위치 및 기간

데이터는 자체 서버에 저장되며, 사용자가 삭제 요청을 하기 전까지 보관됩니다.

5. 데이터 공유

학교와 교육청 ID는 교육청 API를 통해 급식 정보를 가져오는 데 사용됩니다. 그 외의 제3자와의 데이터 공유는 없습니다.

6. 사용자 권리

사용자는 데이터 삭제를 요청할 수 있습니다.

7. 개인정보 처리방침 변경

본 방침이 변경될 경우, DM이나 길드 메시지를 통해 사용자에게 고지할 것입니다.

8. 연락처 정보

본 방침에 대한 문의는 bob@youngzheimer.com으로 연락해주시기 바랍니다.
